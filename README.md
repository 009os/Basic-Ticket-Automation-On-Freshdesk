# Basic Ticket Automation On Freshdesk (College Assignment)

A lightweight Python project for automating basic ticket operations on Freshdesk. Easily manage, create, and update Freshdesk tickets with minimal setup.

## Features

- Automate ticket creation, updates, and management
- Simple and easy-to-use Python scripts
- Designed for quick setup and use with Freshdesk API

## Getting Started

1. **Clone the repo:**  
   `git clone https://github.com/009os/Basic-Ticket-Automation-On-Freshdesk.git`
2. **Install dependencies:**  
   `pip install -r requirements.txt`
3. **Configure your Freshdesk API credentials** in the script or via environment variables.
4. **Run the automation script:**  
   `python ticket_automation.py`
